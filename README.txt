Salon Client Book PWA

What this version adds
- Hair, waxing, eyebrows, and eyelashes categories
- Client-specific service schedules
- Different reminder frequencies for different services and different people
- Due soon / overdue reminder list
- Copyable reminder text for follow-up messages
- CSV export for visits and schedules
- Installable web app files (PWA package)

Important
- This is not an App Store app package.
- It is a phone-ready web app that can be hosted and added to an iPhone Home Screen.
- Data is saved in browser storage on that device unless you export a backup.

How to use
1. Put the whole folder on a simple web host (GitHub Pages, Netlify, Cloudflare Pages, etc.).
2. Open the hosted link in Safari on iPhone.
3. Tap Share > Add to Home Screen.
4. Use Export full backup regularly.

Main workflow
- Add client
- Add service schedule in Reminders (example: Trim every 90 days)
- Log visits in Add Visit
- Dashboard and Reminders will show who is due or overdue
